# Section 1 — Introduction to MySQL Triggers

Section 1 is **concept-only**: it explains what a trigger is, the 6 possible
shapes (`BEFORE/AFTER` × `INSERT/UPDATE/DELETE`), and the difference between
`BEFORE` and `AFTER`.

There is no runnable SQL in this section — just the explanations from
`MySQL-Triggers.md`. Move on to **02-managing-triggers/** for the first
`CREATE TRIGGER` example.
